## 2026-05-19 小程序 timeout 错误修复

### 问题
微信小程序控制台报错：`Error: timeout at Function.<anonymous> (WAServiceMainContext.js)`

### 根因
`pages/index/index.wxml` 中手写死的 `sources` 值与 `app.js` 中定义的不一致：
- wxml 中：`china-holidays` / `lunar-calendar` / `solar-terms` / `yi-ji`
- app.js 中：`holidays` / `lunar` / `solar` / `yiji`
- 实际 GitHub Pages 文件名：`holidays.ics` / `lunar.ics` / `solar.ics` / `yiji.ics`

### 修复
- `pages/index/index.js`：onLoad 中从 `app.globalData.subscriptions` 取数据设到 page data
- `pages/index/index.wxml`：改用 `wx:for="{{subscriptions}}"` 动态渲染，不再手写 sources 值
