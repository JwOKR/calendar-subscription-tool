### 增加 icons 参数控制 emoji 显示
- **提交**：`265aeb2` feat: 增加 icons 参数控制 emoji 显示
- **功能**：URL 参数 `icons=false` 时，所有事件摘要去除 emoji
- **实现**：
  1. 新增 `stripAllEmoji()` 辅助函数（正则覆盖常见 emoji Unicode 范围）
  2. `flattenHolidays()` 支持 `icons` 参数，控制 🎉/💼 前缀
  3. `generateCalendar()` 支持 `icons` 参数，对其他来源事件调用 `stripAllEmoji()`
  4. Handler 读取 `req.query.icons`，默认 `true`，显式传 `false` 才关闭
- **用法**：
  - 默认（有 emoji）：`?sources=holidays,lunar,solar,festivals`
  - 无 emoji：`?sources=holidays,lunar,solar,festivals&icons=false`
- **部署**：已推送 GitHub + Vercel 生产部署完成
